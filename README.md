# 🐞 Mini Bug Tracker

Simple bug tracker built with HTML, CSS, and JS (no framework).  
Ideal for practicing DOM manipulation, form handling, and status filtering.

## 🔧 Features
- Add, update, delete bugs
- Filter by status
- Severity coloring
- Friendly UI
